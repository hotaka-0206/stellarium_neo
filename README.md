# Stellarium NEO

Stellariumで地球接近天体（NEO）を表示し、表示位置を検証するための卒研用リポジトリです。

## 目的

Stellarium上で表示される地球接近天体（NEO）の位置が、JPL Horizonsなどの信頼できるデータとどの程度一致しているかを確認します。

主に、Stellariumで表示される天体の位置と、JPL Horizonsで取得した位置情報を比較することを目的としています。

## 内容

このリポジトリでは、以下の内容を扱います。

- StellariumでNEOを表示する手順
- JPL Horizonsとの比較方法
- 検証用スクリプト
- 調査メモ
- 作業記録

## Stellariumのメモ

### リモートコントロールプラグインの設定

StellariumをPythonなどの外部プログラムから操作するために、リモートコントロールプラグインを有効にします。

### 手順

1. Stellariumを起動する
2. `F2` キーを押して、設定画面を開く
3. `プラグイン` を選択する
4. プラグイン一覧から `リモートコントロール` を選択する
5. `起動時に実行` にチェックを入れる
6. `設定` を開く
7. 以下の項目にチェックを入れる
   - サーバーが利用可能
   - 起動時に自動的に有効にする
8. `設定を保存` を押す
9. Stellariumを再起動する
10. 再起動後も設定が保存されていれば完了


### 惑星の入力形式

例：Jupiter<br>
形式：惑星名

- 頭文字は大文字で入力

### 小惑星の入力形式

例：(1) Ceres<br>
形式：(小惑星番号) 小惑星名<br>

- ()と小惑星名の間は半角スペースを空ける
- コード内で登録された小惑星は惑星同様の入力でfocusできる


## 注意

JPLの天体暦ファイル（例: `DE440.bsp`）は容量が大きいため、このリポジトリには含めません。

必要な場合は、各自でJPLなどからダウンロードしてください。

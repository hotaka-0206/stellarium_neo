\# Stellarium NEO



Stellariumで地球接近天体（NEO）を表示するための卒研用リポジトリ



\## 目的



Stellarium上で表示されるNEOの位置が、JPL Horizonsなどの信頼できるデータとどの程度一致しているかを確認する。



\## 内容



\- Stellariumの設定メモ
-リモートコントロールプラグインを設定
\-1．設定画面(F2)からプラグイン→リモートコントロール．”起動時に実行”にチェックを入れ"設定"
\-2．”サーバーが利用可能”，”起動時に自動的に有効にする”にチェックをいれ”設定を保存”
\-3．stellariumを再起動し,設定内容が保存されていたら完了

\- NEOの表示手順

\- JPL Horizonsとの比較方法

\- 検証用スクリプト

\- 調査メモ



\## 注意



JPLの天体暦ファイル（例: DE440.bsp）は容量が大きいため、このリポジトリには含めません。

必要な場合は、各自でJPLなどからダウンロードしてください。
